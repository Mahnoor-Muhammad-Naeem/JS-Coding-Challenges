# Learn JavaScript ‚Äì A Practical, Challenge‚ÄëFirst Roadmap

This README gives you a hands‚Äëon path to learn modern JavaScript through the **best free + high‚Äëquality resources** and **curated coding challenges**. It‚Äôs split into stages so you can jump in at your current level.

---

## üéØ Learning Goals
- Write clean, idiomatic ES6+ JavaScript
- Master the DOM, events, async (Promises/`async`‚Äì`await`), and modules
- Build small projects, then real apps (with tests)
- Learn just enough Node.js and tooling to be productive

---

## üß∞ Prerequisites & Setup
- **Install Node.js** (use **nvm** to manage versions): <https://github.com/nvm-sh/nvm>
- **Editor:** VS Code + extensions: Prettier, ESLint, GitLens
- **Format & Lint:** Prettier + ESLint (Airbnb or Standard config)
- **Browser DevTools:** Learn the Sources tab, breakpoints, Network panel

---

## üó∫Ô∏è Roadmap (8 Weeks, ~7‚Äì10 hrs/week)

### Week 1 ‚Äì JS Basics & Syntax
- Variables (`let`/`const`), types, operators, conditionals, loops
- **Challenges:**
  - Recreate FizzBuzz
  - Reverse a string, check palindrome
- **Resources:**
  - **MDN JS Guide:** <https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide>
  - **javascript.info (The Modern JS Tutorial):** <https://javascript.info>

### Week 2 ‚Äì Functions, Arrays, Objects
- Functions, arrow funcs, array methods (`map/filter/reduce`), object patterns
- **Challenges:**
  - Implement `map`, `filter` from scratch
  - Frequency counter (anagrams)
- **Resources:**
  - **Eloquent JavaScript (chapters 1‚Äì5):** <https://eloquentjavascript.net/>

### Week 3 ‚Äì DOM & Events
- Querying/manipulating DOM, events, delegation, forms
- **Project:**
  - **Vanilla To‚ÄëDo App** (add/edit/delete, filter, persist to `localStorage`)
- **Resources:**
  - **MDN ‚Äì DOM:** <https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model>

### Week 4 ‚Äì Async & APIs
- Callbacks ‚Üí Promises ‚Üí `async`/`await`, Fetch API, error handling
- **Project:**
  - **Weather Dashboard** (Fetch by city, loading/error states)
- **Resources:**
  - **MDN ‚Äì Fetch:** <https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API>
  - **You Don‚Äôt Know JS Yet ‚Äì Async & Performance (free):** <https://github.com/getify/You-Dont-Know-JS>

### Week 5 ‚Äì Modules, Build, and Patterns
- ES Modules, bundlers (Vite), environment variables
- **Project:**
  - **Quiz App** (module split, state machine, timer)

### Week 6 ‚Äì Testing Fundamentals
- Unit tests with **Jest**, DOM tests with **@testing-library/dom**
- **Challenges:**
  - TDD a simple calculator
  - Test the Quiz App
- **Resources:**
  - **Jest Docs:** <https://jestjs.io/docs/getting-started>
  - **Testing Library:** <https://testing-library.com/docs/dom-testing-library/intro/>

### Week 7 ‚Äì Data Structures & Algorithms (DSA, optional but valuable)
- Time/space complexity, classic problem patterns (two pointers, sliding window)
- **Challenges:** 10‚Äì15 kata on arrays/strings/stacks
- **Resources:**
  - **Codewars:** <https://www.codewars.com/>
  - **LeetCode (Top Interview/Easy‚ÄìMedium):** <https://leetcode.com/>
  - **Exercism ‚Äì JavaScript track:** <https://exercism.org/tracks/javascript>
  - **JSChallenger:** <https://www.jschallenger.com/>

### Week 8 ‚Äì Capstone & Portfolio
- **Pick one:**
  - **Movie Explorer** (search, details, favorites, pagination)
  - **Kanban Board** (drag‚Äëdrop, persistence)
- Add **README**, live demo (Netlify/Vercel), and tests
